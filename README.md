Me chamo Pedro Reis e este repositório foi criado para armazenar três planilhas com dados totalmente fictícios simulando informações corporativas da Natura &Co, empresa do setor ambiental. O objetivo do projeto é oferecer uma base estruturada para prática de análise de dados, criação de dashboards (Power BI, Tableau ou Excel), estudos estatísticos e exercícios de organização e manipulação de planilhas. As planilhas incluem dados simulados sobre funcionários (cargo, departamento, salário e tempo de empresa), projetos ambientais (investimento, região, cronograma e indicadores de impacto) e indicadores de sustentabilidade (emissão de CO₂, consumo de água, energia renovável e reciclagem). Todos os dados são fictícios e possuem finalidade exclusivamente educacional e de portfólio.
